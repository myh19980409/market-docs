查看个人信息 #

请求格式 #

GET /profile?uid=1001

携带cookies，参数 uid 表示用户id，省略时获取自己的信息

返回格式 #

{
    "uid": 1001,
    "username": "张三",
    "phone": "13853838438",
    "realname": "张三",
    "class": "0班",
    "sno": "12345678",
    "dormitory": "1号楼 101",
    "gender": "男",
    "createtime": 1640092818,
}

查看别人的信息不可见的内容返回空值

获取用户发布商品列表 #

GET /item?uid=1001

请求格式 #

携带cookies，参数 uid 表示用户id，省略时获取自己发布的商品

暂时不考虑分页，直接返回全部结果即可

返回格式 #

[
    {
        "id": 114514,
        "uploadtime": 1640000000,
        "name": "",
        "level": 9,
        "remark": "",
        "price": "",
        "sort": "",
        "count": "",
        "display": "",
        "sales": "",
        "uid": 1001,
        "image": "xxxxxxxxxxxxxxxxx.jpg",
    },
    // ...
]
  • id: 主键（商品id）
  • uploadtime: 上传时间
  • name: 商品名称
  • level: 商品成色（几成新）
  • remark: 商品详细信息
  • price: 商品价格
  • sort: 商品类别
  • count: 商品数量
  • display: 商品是否被下架
  • sales: 商品销量
  • uid: 发表的用户id
  • image: 商品的图片地址

（其实是复制的）
